# ESP32_OLED_Weather
  
1.3" oled display 128x64. sh1106 controller.  

3.3V -- VCC  
GND  -- GND  
D4   -- SCL  
D5   -- SDA  
  
It need "config.txt" with requiring following info.  
ssid=(wifi name)  
pass=(wifi pass)  
lat=(your location)  
lon=(your location)  
appid=(openweathermap appid)
